# Khushu 0.24.10

Stability and polish release focused on prayer flow, Tasbih refinement, and Arabic layout fixes.

## Highlights

- Prayer completion now follows the real sequence instead of letting any row be checked at any time.
- Tasbih now supports bundled count sounds and reliable custom prayer alarm sound selection.
- Immersive Arabic and RTL layouts were corrected for Salah, Tasbih, and the bottom navigation indicator.

## Improved

- Merged prayer card polish, smoother sun path visuals, and clearer Home prayer-source status.
- Qibla helper and Home utility actions were refined for the `0.24.10` release candidate.
- Tasbih collection setup and customization flow now behave more naturally.

## Fixed

- Custom prayer alarm sound selection now opens the picker and uses the chosen sound correctly.
- Tasbih and Salah immersive widgets no longer drift out of view under Arabic system layout or RTL conditions.
- Bookmark jumps to the correct ayah and non-Quran readers no longer show Quran-only settings.
